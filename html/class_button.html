void Button::buttonDisplay ( unsigned int  myBorderColor = 0xf800,
unsigned int  myFillColor = 0x0000 
)

Displays the button when pressed.

The default border color is RED and the default fill color is BLACK. Button's border color and fill color is updated. The button is then drawn. After a 100 ms delay, the button is reset back to its original values.

Parameters
myBorderColorThe border color of the button. Default color is red.
myFillColorThe fill color of the button. Default color is black.

< Saves the original fill color
